Description

To quantitatively measure SSBC's website's success, metrics like Key performance indicators are used.
To know more about the indication of the general performance of the established website, you can study more on web analytics. Such examples of softwares would be Google Analytics which can be used to analyze data such as:

  • Volume of visitors to the website
  • Geographical location of visitors
  • Devices/ operating systems that visitors use
  • Where visitors will be directed to and from
  • How users navigate throughout the website
  • The popularity of different pages
  • The conversion rate of the website
